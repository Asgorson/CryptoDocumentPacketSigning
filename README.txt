+-----------------------------------------------------------------------------+
|   Программа для автоматического подписания пакета документов в КриптоПРО.   |
+-----------------------------------------------------------------------------+

Приготовления:

    0. Включить возможность запуска скриптов от имени администратора: <Set-ExecutionPolicy RemoteSigned>
    1. В AD в политике AppLocker "Правила сценариев" добавить пользователя или группу для правила "*\CryptoDocumentPacketSigning.ps1"

Принциа работы:

    0. Расположить папку AutoSigning на рабочем столе.
    1. Файлы для подписания скопировать в папку 'Unsigned'
    2. Папка 'Signed' должна быть пустой
    3. Запустить скрипт 'CryptoDocumentPacketSigning.ps1' 
    5. В консоль ввести отпечаток сертификата подписи. Его можно найти в < Меню пуск -> Сертификаты пользователя -> Личное -> Сертификаты -> "Найти нужный сертификат" -> ПКМ "Свойства" -> Состав -> Отпечаток >
    4. Во время выполнения программы не трогать компьютер и не преключаться на другие окна
